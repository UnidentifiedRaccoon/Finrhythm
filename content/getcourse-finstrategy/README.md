# Курс «ФинCтратегия»: выгрузка GetCourse

Статус: exported with 3 blocked lessons

Источник: `https://fgrm.ncfg.ru/teach/control/stream/view/id/546010026`

Эта папка содержит безопасную markdown-выгрузку видимого контента уроков курса «ФинCтратегия». Сначала были собраны ссылки на дочерние module streams, затем ссылки уроков внутри модулей, после этого exporter открыл каждый lesson URL отдельно. Закрытые уроки были повторно обработаны через авторизованную admin-сессию без передачи пароля в чат или repo files.

Важно:

- видео, архивы и защищённые assets не скачивались;
- по вложениям сохранены только видимые названия и URL;
- по видео сохранены только DOM metadata;
- видимые same-host PDF/DOCX/XLSX вложения были отдельно скачаны в `downloads/`;
- уроки, которые остались недоступны даже через admin-сессию, записаны со статусом `blocked`;
- все финансовые, налоговые, кредитные, инвестиционные и пенсионные утверждения требуют human review.
