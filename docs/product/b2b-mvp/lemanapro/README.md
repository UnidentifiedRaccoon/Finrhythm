# Lemanapro B2B MVP Product Docs

Эта директория содержит canonical product baselines для B2B-first MVP. Binary files in `references/` are supporting artifacts; markdown files below are the operational source-of-truth for agents.

## Canonical docs

- `product-foundation-v1.md` — accepted product-intent baseline.
- `learning-methodology-v0.2.md` — MVP learning, diagnostics, routing and approve-flow baseline with human gates.
- `content-mvp-spec-v0.1.md` — draft content/CMS specification with human gates.
- `design-system-v0.1.md` — draft product-design style baseline for UI implementation, design tokens, component QA, screen generation and visual consistency checks.

## Reference artifacts

- `references/product-foundation-v1.docx`
- `references/lemanapro-comprehensive-program.pptx`
- `references/design-system-style-board-v0.1.png`

`design-system-v0.1.md` is a product/design baseline, not educational content. It must not be used as a source for lesson facts, diagnostics scoring or financial/tax wording.
